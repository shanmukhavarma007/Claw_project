# Landing Page Repository

A simple, responsive landing page built with HTML and CSS.

## Structure

- `index.html` — Main landing page
- `style.css` — Styles (embedded for simplicity)
- `README.md` — This file

## Features

- Clean, modern design
- Responsive layout (mobile-friendly)
- Fast, no dependencies
- Easy to customize

## Usage

Just open `index.html` in a browser. Deploy to GitHub Pages, Netlify, Vercel, or any static host.

## Customization

Edit `index.html` to change content and `style.css` (or the `<style>` block) to tweak the look.
